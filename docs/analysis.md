# Análisis del Acta, Propuesta y OKRs

Este documento resume el análisis del comité evaluador.

Prometido y aprobado:
- Registro digital de siniestros, evidencia, geolocalización, dashboards, reportes, portal ciudadano.

Funcionalidades obligatorias:
- Autenticación, CRUD usuarios, CRUD incidentes, subida de fotos, GPS, generación PDF, dashboard, roles y permisos, auditoría.

Opcionales:
- Integración avanzada de IA (se incluye arquitectura y hooks, no modelos en producción en este MVP).

No desarrollar:
- Microservicios, contenedores, frameworks JS (React/Angular), bases no MySQL.
